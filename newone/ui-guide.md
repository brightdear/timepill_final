# Timepill v3 앱 UI 설명서

---

## 탭 네비게이션 (하단 4개 탭)

| 탭 | 역할 |
|----|------|
| 홈 | 오늘 복용 일정 및 인증 |
| 등록 | 약/슬롯 등록 및 수정 |
| 기록 | 월별 복용 기록 조회 |
| 설정 | 앱 환경설정 |

## 홈 화면 (`app/(tabs)/index.tsx`)

**역할:** 오늘의 복용 스케줄을 한눈에 보여주고, 인증/관리하는 메인 화면

### 표시 요소
- **Timepill** 앱 타이틀
- **스트릭 배지 (가로 스크롤)** — 연속 복용 중인 약의 🔥 현재 스트릭 / 최고 스트릭
- **"오늘 알람"** 섹션 — 당일 등록된 슬롯 목록 (TimeslotRow)
- 슬롯이 없을 때 빈 상태 안내 문구

### 버튼

| 버튼 | 위치 | 동작 |
|------|------|------|
| **약 등록하기** | 빈 상태 | 등록 탭으로 이동 |
| **↑** | 하단 우측 (스크롤 시 등장) | 목록 최상단으로 스크롤 |
| **TimeslotRow 정보 영역** (시간/약이름/정 수) | 각 슬롯 | 해당 슬롯의 수정 화면(등록 탭)으로 이동 |
| **인증** | 각 슬롯 우측 | 스캔 화면(/scan) 열기 |
| **완료** | 인증 완료된 슬롯 | 비활성 표시 (누를 수 없음) |
| **ON/OFF 토글** | 각 슬롯 우측 | 슬롯 활성화/비활성화 다이얼로그 |
| **✕ (삭제)** | 각 슬롯 우측 | 슬롯 삭제 확인 다이얼로그 |

### ON/OFF 토글 다이얼로그

| 선택지 | 동작 |
|--------|------|
| 하루만 건너뛰기 | 인증 창 만료까지만 비활성화 |
| 완전 OFF | 영구 비활성화 |
| 취소 | 닫기 |

### FreezePopup (어제 복용 놓쳤을 때 자동 표시)

| 버튼 | 동작 |
|------|------|
| 사용 안 함 | 닫기 (freeze 소비 안 함) |
| 사용 | 선택된 약에 freeze 적용 → streak 보호 |

여러 약을 놓친 경우 체크박스로 약을 선택 가능 (남은 freeze 수 한도 내)

## 등록 화면 (`app/(tabs)/register.tsx`)

**역할:** 새 약 슬롯 등록 또는 기존 슬롯 수정. 슬롯Id 파라미터 유무에 따라 모드가 전환됨.

### 섹션별 요소 및 버튼

#### 1. 약 이름 (신규 등록 시만 표시)
- 텍스트 입력 (`약 이름 입력` 플레이스홀더)
- 기존 약 이름 드롭다운 제안 → 탭하면 해당 이름 자동 입력

#### 2. 복용 수
- 휠 피커: 1~10 정
- 5정 이상이면 경고 문구: *"복용 수가 많으면 스캔 정확도가 낮아질 수 있습니다"*

#### 3. 시간
- 휠 피커 3개: 오전/오후 | 시(1~12) | 분(00~59)

#### 4. 주기 (CyclePicker)

| 버튼 | 동작 |
|------|------|
| 매일 | 매일 복용 |
| 주중 | 월~금 복용 |
| 주말 | 토~일 복용 |
| 요일 선택 | 요일 개별 선택 (일/월/화/수/목/금/토 토글 버튼 등장) |
| 휴약기 | X일/주 복용 후 Y일/주 휴약 패턴 → 숫자 휠 + 일/주 토글 |

#### 5. 인증 가능 시간

| 버튼 | 동작 |
|------|------|
| 30분 | 알람 시각 ±30분 내 인증 가능 |
| 60분 | ±60분 내 |
| 120분 | ±120분 내 |

#### 6. 알람 설정

| 요소 | 동작 |
|------|------|
| 팝업 알림 토글 | 앱 내 팝업 알림 ON/OFF |
| 알람 토글 | 오디오 알람 ON/OFF |
| 강제 알람 토글 | (알람 ON일 때) 강제 알람 ON/OFF |
| 알람 소리: 기본/벨/차임 | 알람 소리 종류 선택 |
| 진동 토글 | 진동 ON/OFF |
| 스누즈 토글 | 스누즈 ON/OFF |
| 스누즈 횟수 휠 (1~3) | (스누즈 ON일 때) 스누즈 최대 횟수 설정 |

#### 7. 사진 등록 (기준 사진, 최소 3장 ~ 최대 10장)

| 버튼 | 동작 |
|------|------|
| + (추가) | 사진 추가 옵션 다이얼로그 열기 |
| ✕ (각 사진) | 해당 사진 삭제 |

**사진 추가 다이얼로그:**

| 선택지 | 동작 |
|--------|------|
| 카메라로 찍기 | 카메라 오버레이 열기 |
| 갤러리에서 선택 | 갤러리 피커 열기 |
| 취소 | 닫기 |

**카메라 오버레이:**

| 버튼 | 동작 |
|------|------|
| ✕ (좌상단) | 카메라 닫기 |
| 촬영 | 기준 사진 촬영 |

중앙 점선 가이드 프레임 + *"알약이 화면 중앙에 오도록 맞춰주세요"* 안내 표시

#### 8. 하단 고정 버튼

| 버튼 | 동작 |
|------|------|
| **저장** / **저장 중...** | 슬롯 저장 또는 업데이트 |

## 기록 화면 (`app/(tabs)/history.tsx`)

**역할:** 월별 복용 기록, 복용률 통계, 연속 복용 스트릭 조회

### 표시 요소
- 월 네비게이션 (이전/다음)
- 달력 (약별 컬러 점, 점 투명도 = 완료율)
- **이번 달 복용률** — 대형 퍼센트 표시
- **약별 복용률** — 진행 바 + 퍼센트
- **연속 복용** — 현재 스트릭 / 최고 스트릭

### 버튼

| 버튼 | 동작 |
|------|------|
| **<** | 이전 달로 이동 |
| **>** | 다음 달로 이동 |
| **달력의 날짜** | 해당 날 기록 상세 바텀시트 열기 |

### 날짜 상세 바텀시트 (모달)

| 요소 | 동작 |
|------|------|
| 각 복용 기록 행 | 상태 점 + 약 이름 + 복용량 표시 |
| **삭제 버튼** (각 기록) | 삭제 확인 다이얼로그 → 확인 시 기록 삭제 |
| **닫기** | 바텀시트 닫기 |
| 배경 탭 | 바텀시트 닫기 |

## 설정 화면 (`app/(tabs)/settings.tsx`)

**역할:** 앱 전반의 설정 관리

### 요소 및 버튼

| 요소 | 동작 |
|------|------|
| **Private Mode 토글** | ON: 약 이름을 '알약1', '알약2'로 익명 표시 |
| **Freeze 현황 카드** | 현재 freeze 개수 시각 표시 (3칸 중 채워진 수), 읽기 전용 |
| **한국어 / English / 日本語** | 앱 언어 변경 |
| **Dev Mode 토글** | ON: 스캔 결과 디버그 패널 + FP/FN 저장 활성화 |
| **Timepill v1.0.0** | 버전 표시, 인터랙션 없음 |

## 스캔 화면 (`app/scan.tsx`) — 모달

**역할:** 카메라로 알약을 찍어 기준 사진과 비교 인증

### 화면 요소

| 요소 | 설명 |
|------|------|
| 중앙 가이드 프레임 | 흰 점선 정사각형 (화면의 60%) |
| 알약 감지 배지 | 알약 감지 중일 때 하단에 *"알약이 감지되고 있습니다"* |
| 우측 약 칩 목록 | 인증 가능한 약 목록 (세로 스크롤), 선택 시 하이라이트 |
| 스캔 중 로딩 오버레이 | *"알약을 인식하는 중..."* + 스피너 |

### 버튼

| 버튼 | 위치 | 동작 |
|------|------|------|
| **✕** | 좌상단 | 스캔 화면 닫기, 홈 복귀 |
| **⚡ / ⚡ON / ⚡A** | 우상단 | 플래시 모드 순환 (끄기 → 켜기 → 자동) |
| **약 칩** (우측 세로 목록) | 우측 | 스캔할 약 선택 |
| **SCAN** | 하단 중앙 (원형 흰 버튼) | 스캔 실행 |

### 스캔 결과 알림 (Alert)

| 상황 | 메시지 |
|------|--------|
| 성공 | "[약 이름] 인증 완료!" |
| 전체 완료 | "현재 모든 알약을 인증하셨습니다!" |
| 알약 미감지 | "알약이 감지되지 않았습니다" |
| 알약 너무 작음 | "알약이 너무 작습니다" |
| 매칭 실패 | "알약을 확인할 수 없습니다" |

### Dev Mode 스캔 피드백 패널 (Dev Mode ON 시)

| 버튼 | 동작 |
|------|------|
| **정확함** | 올바른 결과로 피드백 저장 |
| **FP** (False Positive) | 오탐(없는데 있다고 판정) 피드백 저장 |
| **FN** (False Negative) | 미탐(있는데 없다고 판정) 피드백 저장 |

## 알람 화면 (`app/alarm.tsx`) — 모달

**역할:** 예약된 시간에 알림이 울릴 때 나타나는 화면

### 표시 요소
- 알람 시각 (HH:MM, 대형)
- 약 이름
- 복용량 (정)

### 버튼

| 버튼 | 동작 |
|------|------|
| **인증하기** (초록) | 알람 닫기 → 스캔 화면으로 이동 |
| **스누즈** (흰 반투명, 스누즈 설정 시) | 5분 후 재알림 예약, 잔여 횟수 표시 |
| **알람 끄기** (투명) | 알람 닫기 + 모든 알림 취소 |

## 강제 알람 화면 (`app/force-alarm.tsx`) — 모달

**역할:** 강제 알람 설정 시 나타나는 강화된 알람 화면 (여러 약 동시 표시 가능)

### 특징
- 여러 약이 동시에 강제 알람인 경우 카드 스택으로 표시
- 각 카드마다 독립적인 버튼

### 카드 당 버튼

| 버튼 | 동작 |
|------|------|
| **인증하기** (초록) | 해당 약 스캔 화면으로 이동 |
| **스누즈** (흰 반투명) | 해당 약 5분 스누즈 예약, 잔여 횟수 표시 |
| **긴급 탈출** (빨강) | 탈출 확인 다이얼로그 열기 |

### 긴급 탈출 확인 다이얼로그

> *"강제알람을 무시하고 종료합니다. 이 행동은 기록됩니다."*

| 버튼 | 동작 |
|------|------|
| 취소 | 닫기 |
| 종료 | 탈출 기록 후 해당 약 제거 |

## 전체 팝업/다이얼로그 요약

| 팝업 | 트리거 | 버튼 |
|------|--------|------|
| FreezePopup | 어제 복용 놓침 감지 | 사용 안 함 / 사용 |
| FreezeAcquiredPopup | 15일 연속 달성 | 확인 |
| 날짜 상세 시트 | 달력 날짜 탭 | 삭제 / 닫기 |
| 사진 추가 다이얼로그 | + 버튼 탭 | 카메라 / 갤러리 / 취소 |
| 슬롯 ON/OFF 다이얼로그 | 토글 버튼 탭 | 하루 건너뛰기 / 완전 OFF / 취소 |
| 슬롯 삭제 다이얼로그 | ✕ 버튼 탭 | 삭제 / 취소 |
| 기록 삭제 다이얼로그 | 삭제 버튼 탭 | 삭제 / 취소 |
| 긴급 탈출 다이얼로그 | 긴급 탈출 버튼 탭 | 취소 / 종료 |
| Dev 피드백 패널 | 스캔 완료 (Dev Mode) | 정확함 / FP / FN |

## 화면 이동 흐름

```
하단 탭
├── 홈 → 등록(수정 모드)
│      → /scan (인증 버튼)
├── 등록 → 카메라 오버레이
├── 기록 → 날짜 상세 바텀시트
└── 설정

모달 (탭 위에 오버레이)
├── /scan → (인증 완료 후) 홈 복귀
├── /alarm → /scan
└── /force-alarm → /scan
```
